How to add a record

Step 1 Getting to the form page

On the top toolbar click the . This will take you to the application form.

Step 2 Choosing a booking type

Select which type of booking you wish to make. the form changes fields based on the booking type

Step 3 Fill in the form

The form has a few required fields, these fields are marked with an astrix (*).

Some fields:
Client: Used as a heading / description for the record

Suggestions Quick way to capture a new record

Instead of manually typing out a whole lot of the fields data click on a "Suggested Accounts" record to set the account number to that account (most recently used accounts).

once an account number is set (either from the drop down or the Suggested Accounts pane) the "Suggestions" pane will apear based on account history for the past 60 days. clicking on any of the buttons will auto populate the forms field with that value.

You also have an option to see the 6 most recent bookings for the selected account
