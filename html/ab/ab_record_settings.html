Changing the record list
* Changes are saved per section you're on. Bookings, Production etc

Step 1 open the settings dropdown

Click the spanner icon to open the list settings options

Options Dropdown

Group By Group the records by these categories

Select any of the group by options, the options change depending on which page you're on.

Direction Ordering the records

ASC - Ascending , order the smallest on top building up to the largest

DESC - Descending , order the largest on top going to the smallest

Settings Settings pane

select this option if you want to make even more changes to your list, for instance adding / re-ordering columns.

Settings Pane

List Columns Change the columns in the list

Left side is the Available columns. these are un-used columns.

Right side are the Selected (used) columns.

Drag the records between the columns to add / remove any from the list. to re arange the order of your columns in list view drag them up and down in the Selected Columns area
