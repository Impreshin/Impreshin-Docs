How to edit a record

Step 1 click the edit button in any details pane

in any details pane near the top next to the client name to the left of the date and publication you should see the edit button clicking it takes you to the edit form. the edit form is similer to the add booking form

Step 2 Make the changes

change the values you wish to change, keep in mind if you wish the total cost to change based on calculation you will have to clear the value first or click the badge next to the field with the correct value in it.

For a list of the fields and their descriptions please see "Form fields"

Step 3 Save the form

Click the button at the bottom of the page

Footer buttons Explanation of the various footer buttons

Delete Button - Clicking it you will be prompted to give a reason for deleting this record

View Record Button - Clicking it will bring up the details pane for this record

Back to list Button - Clicking it will take you to the provisional bookings list

New Record Button - Clicking it clear the form for you to be able to add a new record

Note: all changes are recorded and are visible in the "log" pane in the details window
